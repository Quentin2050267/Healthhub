Accessibility Toolbar Operation Instructions

1. How to Use the Website Browsing Assistive Toolbar

This site provides shortcut keys to turn the accessibility website browsing assistive tool on and off. Additionally, there is an "Accessibility Reading" button in the top navigation bar of the homepage, which can also be used to turn the accessibility website browsing assistive tool on or off.

txt Mode

In txt mode, non-text content such as images on the webpage is converted into text and displayed linearly from top to bottom to meet the needs of people with different access requirements. After entering txt mode, you can also return to normal mode by clicking the "Switch to Visual Mode" button. Additionally, there are corresponding shortcut keys for the txt display mode conversion function, and the use of these shortcut keys does not depend on whether the toolbar is enabled.

Font Size Control

You can adjust the font size of the webpage, including medium font, large font, extra-large font, and restore font functions. The default state of the button is "medium font". By clicking the button, you can change the font size of the page and the button state. The button state changes in the following order: medium font—large font—extra-large font—restore (medium) font.

High Contrast Display

High contrast refers to adjusting the contrast between the foreground content and background color of the current webpage to the maximum, providing help for low vision users to clearly access webpage content.

Assistive Lines

The assistive lines provided by this site are two red baseline lines, horizontal and vertical, which help visually impaired users align their reading position.

Interface Control Functions

Website visitors can adjust the size of the webpage interface according to their needs. Interface zoom in enlarges the display ratio of the current page, and the webpage content will also become larger accordingly; interface zoom out reduces the display ratio of the current page, and the webpage content will also become smaller accordingly.

2. Information Accessibility Function Operation Instructions

1. Open toolbar shortcut key: browser universal shortcut key (Alt+G); quit accessibility system function shortcut key: browser universal shortcut key (Esc).

2. Read dedicated function shortcut key: browser universal shortcut key (Alt+H).

3. Open accessibility help instructions shortcut key: browser universal shortcut key (Alt+?); press the shortcut key (Alt+Q) again to close the help.

4. Refresh page function shortcut key: browser universal shortcut key (Alt+R).

5. Crosshair assistive line function shortcut key: browser universal shortcut key (Alt+C).

6. Open txt mode shortcut key: browser universal shortcut key (Alt+T).

7. Mouse pointer icon shortcut key: change mouse arrow icon, (Alt+M) to change.

8. Enlarge page font shortcut key: browser universal shortcut key (Alt+↑).

9. Reduce page font shortcut key: browser universal shortcut key (Alt+↓).

10. Page color function shortcut key: browser universal shortcut key (Alt+`), select a page style.

11. Voice read function shortcut key: browser universal shortcut key (Alt+V).

12. Speed adjustment shortcut key: adjust speed function, (Alt+S) shortcut key can adjust speed.

13. Point function shortcut key: browser universal shortcut key (Alt+P).

14. Continue function shortcut key: browser universal shortcut key (Alt+X).

15. Magnifier read function shortcut key: browser universal shortcut key (Alt+B).

16. Read dedicated tab function shortcut key: browser universal shortcut key (Tab).

17. Area jump shortcut key: Alt+(1, 2, 3, 4, 5, 6) keys quickly jump to the corresponding area, press the Tab key to browse information.

3. Common Problems and Solutions

1. This website supports all major mainstream browsers and can display well in all major browsers. The website browsing assistive toolbar can display different functions in different versions of browsers. All functions of this website work best in IE.

2. During the website browsing process, if the page is too large and not fully loaded, shortcut keys may fail or error messages may appear. In this case, please refresh and wait patiently for the page to fully load before browsing normally.

3. When using screen reader software to browse the website, if you encounter a captcha, please move the focus to the captcha and press Enter to listen to the audio captcha information.

4. During the website browsing process, if the current webpage cannot be "saved as...", you can use another browser to save it, or save it directly as a ".TXT" file.

5. During the website browsing process, if shortcut keys fail or there are numeric links, it may be because the current page is external content and not within the scope of this site's construction.
